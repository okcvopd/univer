Прогрмма создаёт сервер с хронящейся в ней базой данных с различными журналами, которые были сгенерированны случайным образом. 

./server a.db - запускает сервер с базой данных

./client - запускает клиента

Клиент может использовать следующие команды:

select [фильтры] [from #number] - выводит строки, которые удволетворяют фильтрам, from #number позволяет выводить результаты из прошлых запросов

Фильтры: author=[Автор] - выводит определённого автора;
	 year=[первый год, второй год] - выводит журналы, которые выходили в промежуток между первым годом и вторым;
	 number=[a,b] - выводит те, у которых номер журнала лежит между a и b;
	 
Если написать два фильтра одинакового типа, то получается их объединение, а если разного, то пересечение(см. protocol.txt)

delete [фильтры] - удаляет строки, которые удволетворяют фильтрам

insert - вставляет в базу данных новую строку с заданными в любом порядке, но всеми данными, т.е. article, author, journal, number, year

save - сохраняет базу данных в нынешнем виде
